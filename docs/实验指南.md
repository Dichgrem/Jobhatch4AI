## 第5章 大模型配置与使用

### 5.1 理论学习

见课件：第5章 大模型.pptx 及配套 大模型.ipynb

### 5.2 配置大模型API

1. 登录火山引擎方舟控制台 (https://console.volcengine.com)
2. 进入 API Key 管理，创建 API Key
3. 配置环境变量 `ARK_API_KEY`：
   - Windows: 系统环境变量中新增 `ARK_API_KEY`，值为上一步的Key
   - 验证: `echo %ARK_API_KEY%`
4. 创建自定义推理接入点，获取 Endpoint ID（或 Model ID）

测试代码：

```python
import os
from openai import OpenAI

deepseek_access_point = "your-endpoint-id"  # 替换为自己的
api_key = os.environ.get("ARK_API_KEY")
base_url = "https://ark.cn-beijing.volces.com/api/v3/"

client = OpenAI(base_url=base_url, api_key=api_key)

completion = client.chat.completions.create(
    model=deepseek_access_point,
    messages=[
        {"role": "system", "content": "请使用友好的语气说话"},
        {"role": "user", "content": "你是谁？"},
    ]
)
print(completion.choices[0].message.content)
```

## 第6章 Streamlit构建系统

### 6.1 Streamlit简介

Streamlit 是一个纯Python的Web应用框架，无需前端开发经验。特点：
- 几行代码即可创建功能齐全的Web应用
- 内置数据可视化、交互控件和布局选项
- 兼容 Pandas、Matplotlib、Plotly、PyTorch 等

### 6.2 实操

```bash
streamlit run app.py
```

项目文件：
- `app0.py` - 10行代码实现简单前后端
- `app.py` - 100行代码实现复杂前后端
- `app_main.py` - 面向对象方式实现，包含大模型调用

#### Streamlit登录界面集成

```python
from login.auth import Authenticator

self.login = Authenticator(userfile_path="./data/users.json")

def _check_authentication(self):
    return self.login.auth_page()

def run(self):
    if self._check_authentication():
        self._render_sidebar()
        if self.menu_selection == "数据可视化":
            self._render_data_visualization()
        elif self.menu_selection == "即时聊天":
            self._render_chat()

def _render_sidebar(self):
    with st.sidebar:
        st.write(f"当前用户：{st.session_state.username}")
        self.menu_selection = st.radio(
            "选择功能", ["数据可视化", "即时聊天"], label_visibility="collapsed"
        )
        if st.button("退出"):
            st.session_state.authenticated = False
            st.session_state.username = None
            st.rerun()
```

## 第7章 常见错误

**问题**: `pip install volcengine-python-sdk[ark]` 报目录不存在错误

**解决1**: 注册表中启用长路径：
- `Win+R` → `regedit`
- 路径 `\HKEY_LOCAL_MACHINE\SYSTEM\CurrentControlSet\Control\FileSystem`
- 设置 `LongPathsEnabled` 为 1

**解决2**: 源码安装：
```bash
pip install https://pypi.tuna.tsinghua.edu.cn/packages/.../volcengine-python-sdk-*.tar.gz
# 或
python setup.py install --user
```

# 项目补充说明

## 项目背景

随着互联网招聘的快速发展，各大招聘平台每日涌动着海量信息。多数招聘网站的数据分析和可视化功能有限，无法满足用户对市场趋势和人才需求的深度洞察。用户往往需要花费大量时间收集和整理信息，影响决策效率和准确性。本系统旨在提升招聘信息筛选效率，帮助用户把握就业市场趋势，为企业优化招聘策略提供数据支持。

## 项目内容

本招聘数据智能系统基于以下技术栈：
- **数据爬取**: 八爪鱼采集器
- **数据分析**: Pandas清洗、分析、可视化
- **机器学习**: Kmeans聚类、神经网络分类
- **大模型**: API智能客服及智能分析
- **系统框架**: Streamlit

系统主要功能模块：
1. 数据爬取模块：不少于15000条招聘数据
2. 数据预处理模块：数据清理、变换、规约
3. 数据可视化模块：薪资/学历/企业分布/词云
4. 机器学习模块：NLP文本表示 + 聚类 + 分类
5. AI智能交互模块：基于大模型的求职建议
6. Streamlit系统：登录注册、可视化、智能客服
