# Jobhatch4AI 开发文档

## 快速开始

### Nix 环境（推荐）

```bash
nix develop                          # 进入开发环境
cd src/frontend && bun install       # 前端依赖
cp src/backend/.env.example src/backend/.env  # 配置 API key
just start                           # 启动前后端
```

浏览器访问 `http://localhost:5173`，默认账号 `admin` / `admin123`。

### 非 Nix 环境

#### 1. 安装依赖

**Python (3.11+)**:

```bash
# 方式 A：uv（推荐）
uv sync --config uv-cpu.toml

# 方式 B：pip
pip install fastapi uvicorn pydantic openai python-dotenv python-multipart \
            pandas numpy scikit-learn jieba torch pytest httpx
```

**Bun** (前端构建/运行):

```bash
# macOS / Linux
curl -fsSL https://bun.sh/install | bash

# 安装前端依赖
cd src/frontend && bun install
```

#### 2. 配置

```bash
cp src/backend/.env.example src/backend/.env
# 编辑 src/backend/.env，填入 LLM_API_KEY
```

`.env` 说明：

| 变量 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `LLM_API_KEY` | ✅ | - | API 密钥 |
| `LLM_BASE_URL` | | `https://api.deepseek.com` | API 地址 |
| `LLM_MODEL` | | `deepseek-chat` | 模型名 |

> 兼容火山方舟：设置 `LLM_BASE_URL=https://ark.cn-beijing.volces.com/api/v3`，`LLM_MODEL=你的endpoint-id`。

#### 3. 启动

```bash
# 终端1：启动后端
cd src/backend
uvicorn main:app --reload --port 8000

# 终端2：启动前端
cd src/frontend
bun dev
```

浏览器访问 `http://localhost:5173`。

### 处理大 CSV 数据

项目已包含 93,602 条招聘数据（`data/上市公司招聘数据2026.csv`，受 .gitignore 保护不上传 Git）。

首次使用需上传并处理：

```bash
# 方式1：API 上传
curl -X POST http://localhost:8000/api/data/upload \
  -F "file=@data/上市公司招聘数据2026.csv"

# 方式2：Python 直接处理
cd src/backend && python3 -c "
from app.services.pipeline import run_pipeline
run_pipeline('../../data/上市公司招聘数据2026.csv')
"
```

处理完成后 Dashboard 即时展示真实统计图表。

## 架构

```
src/
├── backend/                        # FastAPI (Python 3.12)
│   ├── main.py                     # 入口，CORS，挂载路由
│   └── app/
│       ├── config.py               # LLM_API_KEY (.env 导入), 模型端点
│       ├── routers/
│       │   ├── auth.py             # /api/auth/{login,register,logout}
│       │   ├── chat.py             # /api/chat (SSE 流式 LLM + RAG 上下文)
│       │   └── data.py             # /api/data/{summary,salary,upload,cluster,...}
│       └── services/
│           ├── auth.py             # SQLite 用户管理
│           ├── database.py         # SQLite WAL (users, auth_tokens, pipeline_state)
│           ├── llm.py              # OpenAI 客户端封装
│           ├── preprocess.py       # jieba 分词，词表，BOW，技术术语
│           ├── classifier.py       # PyTorch MLP 分类器 + 训练/保存/预测
│           ├── clustering.py       # sklearn KMeans 聚类
│           ├── recommender.py      # 技能推荐（余弦相似度）
│           ├── rag.py              # RAG 提示构建，流式生成
│           └── pipeline.py         # CSV→统计 全管道
│
└── frontend/                       # Vue 3 + Bun + Vite
    └── src/
        ├── views/
        │   ├── Login.vue           # 登录/注册
        │   ├── Dashboard.vue       # 数据可视化+词云+Kmeans聚类
        │   └── Chat.vue            # AI 对话
        └── api/index.ts            # fetch 封装
```

## 数据流

```
CSV 上传 → POST /api/data/upload
         → pipeline.run() 分词/词表/统计
         → SQLite pipeline_state 表 持久化
         → GET /api/data/* 返回 Dashboard
```

## 聚类

```
POST /api/data/cluster → KMeans(BOW, 30000样本, 6类)
                       → SQLite (key='kmeans')
                       → Dashboard 聚类卡片
```
