# Jobhatch4AI 开发文档

## 快速开始

### 1. 安装依赖

需要 **Python 3.11+** 和 **uv**：

```bash
# CPU 版（推荐）
just sync

# GPU 版（CUDA 12.1）
just sync-gpu
```

### 2. 配置

```bash
cp src/backend/.env.example src/backend/.env
# 编辑 src/backend/.env，填入 LLM_API_KEY
```

| 变量 | 必填 | 默认值 | 说明 |
|------|------|--------|------|
| `LLM_API_KEY` | ✅ | - | API 密钥 |
| `LLM_BASE_URL` | | `https://api.deepseek.com` | API 地址 |
| `LLM_MODEL` | | `deepseek-chat` | 模型名 |

> 兼容火山方舟：`LLM_BASE_URL=https://ark.cn-beijing.volces.com/api/v3`

### 3. 启动

```bash
just start                           # 同时启动前后端
# 或分别启动
just backend                         # :8000
just frontend                        # :5173
```

浏览器访问 `http://localhost:5173`，默认账号 `admin` / `admin123`。

## 数据导入

项目包含 93,602 条招聘数据（`data/上市公司招聘数据2026.csv`，受 .gitignore 保护）。

```bash
# API 上传
just upload

# 或直接 Python 处理
uv run python -c "
from app.services.pipeline import run_pipeline
run_pipeline('../../data/上市公司招聘数据2026.csv')
"
```

## 嵌入管道

BGE-M3 文本嵌入 + 向量索引 + t-SNE 可视化：

```bash
just embed 5000          # 生成 5000 条嵌入 + FAISS 索引
just tsne 3000 30        # t-SNE 降维
just search "Python" 5   # 语义搜索
just chat "薪资水平"      # AI 对话（含向量检索）
```

## 架构

```
src/
├── backend/                        # FastAPI (Python 3.12)
│   ├── main.py                     # 入口，CORS，挂载路由
│   └── app/
│       ├── config.py               # 环境变量
│       ├── routers/
│       │   ├── auth.py             # /api/auth/*
│       │   ├── chat.py             # /api/chat (SSE + 语义检索)
│       │   └── data.py             # /api/data/*（含嵌入、搜索、分类）
│       └── services/
│           ├── auth.py             # SQLite 用户管理
│           ├── database.py         # SQLite WAL
│           ├── llm.py              # OpenAI 客户端
│           ├── preprocess.py       # jieba 分词 + BOW
│           ├── classifier.py       # MLP + EmbeddingMLP 分类器
│           ├── clustering.py       # KMeans 聚类
│           ├── recommender.py      # 技能推荐
│           ├── embedding.py        # BGE-M3 文本嵌入
│           ├── vector_search.py    # numpy 向量索引 + 语义搜索
│           ├── tsne_viz.py         # t-SNE 降维可视化
│           ├── rag.py              # RAG 提示构建（统计+语义）
│           └── pipeline.py         # CSV→统计 全管道
│
└── frontend/                       # Vue 3 + Bun + Vite
    └── src/views/
        ├── Login.vue
        ├── Dashboard.vue           # 数据可视化 + 词云 + 聚类
        └── Chat.vue                # AI 对话
```

## 常用命令

| 命令 | 作用 |
|------|------|
| `just sync` | 安装依赖（CPU torch） |
| `just start` | 启动前后端 |
| `just stop` | 停止服务 |
| `just test` | 全量测试 |
| `just test-quick` | 测试（跳过嵌入模型） |
| `just embed 5000` | 生成嵌入 + 向量索引 |
| `just tsne` | t-SNE 坐标 |
| `just search "关键词"` | 语义搜索 |
| `just chat "问题"` | AI 对话 |

## 技术栈

| 层 | 技术 |
|----|------|
| 前端 | Vue 3, vue-router, Vite, Bun |
| 后端 | FastAPI, Uvicorn, Pydantic |
| NLP | jieba, sentence-transformers (BGE-M3) |
| ML | PyTorch MLP, scikit-learn KMeans, TSNE |
| 向量检索 | numpy 余弦相似度 |
| LLM | OpenAI SDK → DeepSeek / 火山方舟 |
| 存储 | SQLite WAL |
| 环境 | uv, just |
| 测试 | pytest, ruff, biome |
