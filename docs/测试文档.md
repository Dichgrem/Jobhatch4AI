# 测试文档

## 运行

```bash
just sync               # 安装依赖
just test               # 全量测试
just test-quick         # 跳过嵌入模型测试
just test-file classifier  # 单个测试文件
```

## 结构

```
tests/
├── conftest.py                  # sys.path + temp_db 夹具
├── test_auth.py                 # 用户认证（11）
├── test_preprocess.py           # 分词/词表/BOW/术语（15）
├── test_classifier.py           # BOW MLP 分类器（8）
├── test_recommender.py          # 技能推荐/余弦相似度（7）
├── test_rag.py                  # RAG 提示构建（5）
├── test_pipeline.py             # CSV 管道（5）
├── test_api.py                  # HTTP 端到端（14）
├── test_embedding.py            # BGE-M3 嵌入（5）
├── test_vector_search.py        # 向量索引+搜索（3）
├── test_tsne.py                 # t-SNE 降维（5）
└── test_embedding_classifier.py # 嵌入法 MLP（9）
```

## 覆盖范围

| 模块 | 测试数 | 内容 |
|------|--------|------|
| Auth | 11 | 密码哈希、注册/登录/登出、token 管理 |
| Preprocess | 15 | jieba 分词、词表构建、BOW 向量化、技术术语提取 |
| Classifier | 8 | MLP 初始化/前向传播/预测、训练收敛、eval 确定性、技能统计 |
| Recommender | 7 | 技能推荐、余弦相似度 |
| RAG | 5 | 提示构建、文档加载 |
| Pipeline | 5 | CSV 解析、薪资/学历/经验统计、状态存取 |
| API | 11 | 健康检查、注册/登录、数据摘要/薪资/学历/分析/分类、聊天上下文 |
| Embedding | 5 | BGE-M3 编码形状、归一化、保存/加载 |
| Vector Search | 3 | 索引构建、搜索、文本检索 |
| t-SNE | 5 | 降维坐标、标签、小数据集、保存/加载 |
| Embedding MLP | 9 | 初始化/前向/预测/训练收敛/保存加载 |

> Chat SSE 端点（3 个）标记为 skip，需要 API key。

## 测试模式

- **单元测试**：纯函数，不依赖外部服务（auth, preprocess, recommender, rag）
- **模型测试**：PyTorch MLP 结构验证 + 小样本训练（classifier, embedding_classifier）
- **嵌入测试**：BGE-M3 编码 + 向量搜索 + t-SNE 降维
- **集成测试**：FastAPI TestClient 模拟 HTTP 请求（api）
- **管道测试**：CSV → pipeline.run() → 输出验证（pipeline）
